Tutorial

Step 1: Understanding the Interface expand_more

The main view shows the graph. On the left panel, you can find the input fields for your equations. Try typing 'y = x^2' to see a parabola.

Step 2: Adding Sliders expand_more

To make your graph interactive, use variables. For example, type 'y = a*x + b'. Desmos will automatically offer to create sliders for 'a' and 'b'.

Step 3: Exploring Advanced Functions expand_more

You can also plot trigonometric functions like 'sin(x)' or polar coordinates. Check the official Desmos documentation for a full list of capabilities.
